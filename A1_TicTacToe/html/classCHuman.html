bool CHuman::makeMove ( )

Function that must implement the move of a human player.

Returns
Return value is either 1 or 0, representing whether a game has been won or not after the move, respectively.

The human player is allowed to enter the valid coin position through input of row and column indices. The human player can also terminate the game at any point of time.
